# creative coding bookclub

The home of the [creativecodingbook.club](https://creativecodingbook.club) web app.

## Keyboard Shortcuts

### Sketch Editor
- **Ctrl + Enter**: Execute code 
- **Ctrl + .**: Stop code execution
- **Ctrl + ,**: Show / hide code 
- **Ctrl + ;**: Show / hide console 
- **Ctrl + F**: Format code
- **Ctrl + /**: Toggle comment

### Sketch Manager
- **Ctrl + S**: Save current sketch
